# Workshop website (static)

## Folder structure
- index.html
- overview.html
- key-info.html
- cfp.html
- program.html
- organizers.html
- accessibility.html
- contact.html
- assets/
  - css/styles.css
  - js/site.js
  - img/
    - MainImage.png
    - AlexImage.jpeg
    - MichaelImage.jpeg

## Images
Put your images in `assets/img/` with the exact filenames above.
If an image is missing, the site automatically shows a generated green placeholder.

## GitHub Pages
Commit everything to your repo and enable GitHub Pages from the repo settings.
Use the root folder (or /docs) depending on your repo setup.
